# CLAUDE.md

## Plugin Overview

Exa web search plugin for GUI Chat applications. AI-powered web search using Exa API.

## Common Guidelines

For standard plugin development guidelines, see:
https://github.com/receptron/GUIChatPluginTemplate/blob/main/CLAUDE.md

## Plugin-Specific Notes

### Features
- AI-powered web search
- Semantic search capabilities
- Display search results with links

### External Services
- **Exa API**: Requires API key for search functionality

### Dependencies
- `gui-chat-protocol`: Core protocol for GUI Chat plugins
